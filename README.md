# Sentiment Analysis API

A lightweight machine learning web application and API that predicts whether a movie or product review is positive or negative. Built with Python, Flask, and Scikit-Learn, and deployed on Render.

## 🚀 Live Demo

You can test the application live here:  
👉 **[https://sentiment-analysis-api-pcyh.onrender.com](https://sentiment-analysis-api-pcyh.onrender.com)**

---

## 🛠️ Project Structure

*   `app.py`: The core Flask application that handles web routing and API requests.
*   `requirements.txt`: List of Python dependencies required to run the application.
*   `vector.pkl`: The trained TF-IDF vectorizer used to transform text data.
*   `sentiment.pkl`: The trained Logistic Regression model used for predicting sentiment.
